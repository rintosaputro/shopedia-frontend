## üìå How To Install?

- Clone This Repo

```
git clone https://github.com/rintosaputro/shopedia-frontend
```

- Go To Folder Repo

```
cd shopedia-frontend
```

- Install Module

```
npm install
```

- Type ``` npm run dev ``` To Start The Project
